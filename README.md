# Golf Gym Tracker

An 8-week, 3-day-per-week golf-focused strength and conditioning programme built as a mobile-first web app. Designed for progressive overload with pre-populated exercises, sets, reps, and weights that you can track and adjust in real time at the gym.

## Live App

**https://markhigginsiss.github.io/golf-gym-tracker/**

Add to your iPhone Home Screen via Safari > Share > Add to Home Screen for a full-screen app experience.

## Programme Overview

The programme is built around compound lifts with golf-specific core and rotational work, progressing through four phases over 8 weeks:

| Weeks | Phase | Sets x Reps | Intensity |
|-------|-------|-------------|-----------|
| 1-2 | Hypertrophy Base | 3x12 | 65-67% |
| 3-4 | Hypertrophy Build | 3x10 | 72-75% |
| 5-6 | Strength | 4x8 | 78-80% |
| 7 | Peak Strength | 4x6 | 85% |
| 8 | Deload & Test | 3x8 | 60% |

## Training Days

### Day A — Squat + Push
Back Squat, DB Chest Press, Cable Woodchop, Lat Pulldown, Face Pull, Pallof Press

### Day B — Deadlift + Press
Deadlift, Overhead Press, DB Row, Hip Thrust, Lateral Raise, Leg Curl

### Day C — Legs + Arms
Goblet Squat, Leg Press, Chest Fly, Rear Delt Fly, Tricep Pushdown, Bicep Curl, Calf Raise

## Each Session Includes

- **Activation warm-up** (5 min): Glute Bridges, Band Pull-Aparts, Cat-Cow + T-Spine Rotation, Band Walks
- **Main lifts**: Progressive overload with suggested weights that scale automatically each week
- **Ski Erg finisher**: Intervals progressing from 6x30s hard/90s easy (week 1) to 20 min steady at uncomfortable pace (week 8)

## Features

- Dark theme mobile-first UI
- Tap to mark sets complete
- Editable weight fields — adjust on the fly and it saves automatically
- Progress bar showing sets completed per session
- Week and day navigation
- All data persists in localStorage on your device
- Works offline once loaded

## Golf-Specific Focus

- Rotational power (Cable Woodchops, Pallof Press)
- Hip drive and glute strength (Hip Thrusts, Squats, Deadlifts)
- Scapular stability and shoulder health (Face Pulls, Band Pull-Aparts)
- Anti-rotation core strength for swing consistency
- Conditioning via Ski Erg for cardiovascular base without impact

## Tech

Single HTML file, no dependencies, no build tools. Pure HTML/CSS/JS hosted on GitHub Pages.
